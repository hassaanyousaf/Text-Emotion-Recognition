## Text-Based Emotion Recognition Web App
Classifies text into **12 emotions** from transcribed speech using your trained **TF-IDF + Logistic Regression** model (80k rows).

## **Features**
• 80K+ row dataset • TF-IDF + Logistic Regression (87%+ accuracy)
• Instant prediction with confidence scores
• Clean Streamlit web interface
• Production-ready (model.pkl + tfidf.pkl)

## **Live Demo**: Enter any sentence -> Get emotion instantly!

## **Tech Stack**
Python • scikit-learn • Streamlit • pandas 

## **Run locally**: 
`streamlit run app.py`
## Performance
- **Dataset**: 80,000+ rows
- **Split**: 70/30 stratified
- **Metrics**: F1-score, precision, recall, confusion matrix
